## ✨ Fitur Aplikasi
- 🎀 **AppBar** dengan warna pink pastel.  
- 🎶 **Judul aplikasi**: `Pemutar Musik`.  
- 🎧 **Bottom Control Bar** dengan ikon:
  - 🔀 Shuffle  
  - ⏮ Previous  
  - ▶ Play (ikon utama berukuran besar)  
  - ⏭ Next  
  - 🔁 Repeat  
- 🌷 **Tema warna pastel** (pink soft, pink pastel, pink gelap).
